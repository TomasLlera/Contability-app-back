# Contability App — Backend

API REST para la aplicación de contabilidad. Gestiona locales, rubros, subrubros, movimientos e importación de Excel.

## Demo

[https://contability-app-back.onrender.com](https://contability-app-back.onrender.com)

## Stack

- Node.js + Express
- MongoDB Atlas (Mongoose)
- XLSX para importación/exportación de Excel
